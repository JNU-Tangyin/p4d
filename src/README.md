# `src/` 文件夹说明

`src/` 文件夹包含了本项目的核心 Python 源代码。这些模块封装了数据处理、模型定义和评估逻辑，旨在提供可复用和模块化的功能。

---

## 文件列表及功能说明

-   `__init__.py`
    -   **功能**: 将 `src/` 目录标记为一个 Python 包，允许从其他目录导入其内部模块。

-   `data_preprocessing.py`
    -   **功能**: 负责所有数据加载、清洗、特征工程和数据分割的逻辑。它确保了数据以统一的格式提供给模型。

-   `evaluation.py`
    -   **功能**: 包含了模型评估所需的所有函数和类。它提供了针对分类任务（如准确率、F1分数、AUC）和回归任务（如R²、RMSE、MAE）的标准化指标计算方法。

-   `symbolic_regression.py`
    -   **功能**: 封装了基于 PhySO 框架的符号回归模型。它处理模型的训练、表达式提取和预测逻辑，是本项目的核心模型实现。
