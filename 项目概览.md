# PhySO 符号回归投资决策模型

基于 PhySO 框架的符号回归机器学习项目，用于建设项目投资决策分析和利润预测。

## 项目简介

本项目使用符号回归（Symbolic Regression）技术，从数据中自动发现可解释的数学公式，用于解决两类任务：

- **分类任务**：判断是否应投资某建设项目（是/否）
- **回归任务**：预测项目预期利润额（万元）

符号回归的优势在于能够生成人类可读的数学表达式，而非黑盒模型，这对于需要可解释性的投资决策场景尤为重要。

## 快速开始

### 环境要求

- Python >= 3.8
- PyTorch >= 1.9.0
- PhySO >= 1.1.0

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行实验

**回归任务（默认）**：
```bash
python main.py --task_type regression --epochs 15
```

**分类任务**：
```bash
python main.py --task_type classification --epochs 15
```

**运行经典模型对比**：
```bash
python main.py --task_type regression --compare-classical
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--dataset_name` | investment_decision | 数据集名称 |
| `--task_type` | regression | 任务类型：classification 或 regression |
| `--epochs` | 15 | 训练轮数 |
| `--test_size` | 0.2 | 测试集比例 |
| `--random_state` | 0 | 随机种子 |
| `--compare-classical` | False | 是否运行经典模型对比 |
| `--no-sensitivity` | False | 是否跳过敏感性分析 |

## 核心功能

### 1. 符号回归建模

使用 PhySO 框架自动发现数据中的数学公式：

```python
from src.symbolic_regression import SymbolicRegressionModel

model = SymbolicRegressionModel(seed=0)
model.fit(X_train, y_train, X_names=X_names, epochs=15)
expression = model.get_best_expression()
```

### 2. 线性校准

针对回归任务，支持后线性校准以提升预测精度：

```python
# 未校准 R²: 0.2190
# 校准后 R²: 0.7840
```

### 3. 模型对比

支持与多种经典机器学习模型对比：

- **回归**：Linear, Ridge, Lasso, Decision Tree, Random Forest, SVR, KNN, MLP
- **分类**：Logistic Regression, Random Forest, SVM, Gradient Boosting

### 4. 可视化分析

- ROC 曲线、KS 曲线（分类任务）
- 散点图、残差分析（回归任务）
- 特征敏感性分析
- 模型可解释性对比

## 数据集

### 项目投资决策数据集

- **文件**：`data/项目数据收集表v2.0.xlsx`
- **样本数**：若干建设项目数据
- **特征数**：26个数值特征
- **目标变量**：
  - 分类：是否应投资该项目（是/否）
  - 回归：预期利润额（万元）
